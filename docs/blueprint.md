# **App Name**: Zenith Harrai Shop

## Core Features:

- Product Display: Showcase available in-game products like diamonds and UC with relevant images and descriptions.
- Buy Now Action: Button to navigate to a payment endpoint, for topping up specific product packages, with a clear call to action for purchasing.
- Featured Banner: Display a large, promotional banner at the top of the home screen showcasing current deals or new products.
- User Notification: The bell-shaped icon in the corner triggers user notification such as promotions and order statuses

## Style Guidelines:

- Primary color: Dark Blue (#0A2A4A) to resemble a digital product offering diamond in popular mobile games.
- Background color: Very Dark Gray (#121212) to maintain the gaming aesthetic and the focus on available product.
- Accent color: Bright Cyan (#30D5C8) for active CTAs like 'Buy Now' buttons.
- Headline font: 'Poppins' (sans-serif) for a precise, contemporary feel; all upper case, slightly bolder.
- Body font: 'Inter' (sans-serif) for UI labels and descriptive text for readability.
- Simple, outline-style icons for UI elements (e.g., menu, notifications).
- Use rounded corners for product cards to maintain a consistent design and the banner should always take the above half of the viewport.
